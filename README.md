# Phishing Email Detection Model

## 📌 Project Overview

This project is a Machine Learning-based Phishing Email Detection System built using Python and Scikit-learn.

The model analyzes email/text content and classifies messages as:

- ✅ Safe
- ⚠️ Phishing/Spam

---

## 🚀 Features

- Email text classification
- TF-IDF feature extraction
- URL and keyword analysis
- Machine Learning model using Scikit-learn
- Accuracy and confusion matrix visualization
- Custom email testing

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📂 Dataset

Dataset used:
- SMS Spam Collection Dataset

Contains:
- Ham (Safe messages)
- Spam (Phishing/Spam messages)

---

## 📊 Model Used

- Multinomial Naive Bayes
